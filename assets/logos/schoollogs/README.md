# School Logo Directory

This directory should contain the school logo files for A.s.higher secondary school.

## Required Files:
- `logo.png` - Main logo (recommended: 200x60px for header, 100x30px for footer)
- `logo-small.png` - Small version for mobile (recommended: 40x40px)
- `logo-white.png` - White version for dark backgrounds (optional)

## Current Usage:
- Header navigation: `assets/logos/schoollogs/logo.png`
- Login page: `assets/logos/schoollogs/logo.png`
- Footer: `assets/logos/schoollogs/logo.png`
- Admin panel: `assets/logos/schoollogs/logo.png`

## Notes:
- All logos are referenced in the HTML templates
- PNG format with transparent background recommended
- Place your actual logo files in this directory
